# 安装Git

## 在Linux上安装Git
```
sudo yum install git
```

## 在Ubuntu上安装Git
```
sudo apt-get install git
```

## 在Windows下安装是最坑的

[这里有离线安装包](http://pan.baidu.com/s/1c2A4UTa)
下载后直接点击`GitHub.application`安装即可
